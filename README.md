## 关于刷题✍️

**算法和数据结构** 是编程的基础。如果你喜欢编程，希望在这条路上越走越远并成为技术大🐂，那么刷题可以帮助你：

- 夯实基础
- 锻炼思维
- 拿到大公司的 **offer**
- ...

总而言之，如果想在编程的技术上有所精进，那么刷题无疑是个好办法。

当然，只顾着闷头刷题可能成效并不是很大，建议大家按照分类进行刷题，效果会更好。
